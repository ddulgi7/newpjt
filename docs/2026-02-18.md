### Godot 4 프로젝트 세팅
- **project.godot** 생성
  - Godot 4.3 Mobile 렌더러
  - Android 세로 해상도 480x854, keep_height 스트레치
  - 입력 매핑: WASD/방향키 (move_up/down/left/right), Space/Enter (interact), ESC (cancel), Tab (menu)
  - 오토로드: Events (시그널 버스), GameManager (전역 상태)
  - 물리 레이어: Player, NPCs, Obstacles, Interactables
  - 텍스처 필터: Nearest (픽셀아트용)
  - ETC2/ASTC 압축 활성화 (Android)
- **폴더 구조** 생성: scenes/, scripts/, assets/, data/, addons/, tests/
- **핵심 스크립트**
  - `events.gd` — 시그널 버스 (player, battle, monster, UI, game state 시그널)
  - `game_manager.gd` — GameState enum, 플레이어 데이터, 파티 관리, 세이브/로드, 씬 전환
  - `player.gd` — CharacterBody2D 기반 4방향 탑다운 이동, RayCast2D 인터랙션
  - `player.tscn` — Player 씬 (Sprite2D, CollisionShape2D, InteractionRay, Camera2D)
  - `main.tscn` — 메인 씬 (Player 인스턴스)
- **게임 데이터 JSON**
  - `monsters.json` — 3종 스타터 몬스터 (fire/water/grass), 스탯/기술/진화 스키마
  - `items.json` — 4종 기본 아이템 (포션, 포획볼, 해독제, 부활)
- **기타**: .gitignore, export_presets.cfg (Android), icon.svg
